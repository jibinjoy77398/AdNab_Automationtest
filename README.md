# AdNab Automation Test

## Project Description
AdNab Automation Test is a project designed to streamline the testing process of AdNab applications. This repository contains scripts and framework to facilitate automated testing.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/jibinjoy77398/AdNab_Automationtest.git
   cd AdNab_Automationtest
   ```

2. **Install dependencies**:
   - Ensure you have [Node.js](https://nodejs.org/) installed.
   - Run the following command to install necessary packages:
     ```bash
     npm install
     ```

## Dependencies
- Node.js >= 14.x
- Jest for testing
- Some other additional libraries (see `package.json` for details)

## Commands to Run
- **Run all tests**:
  ```bash
  npm test
  ```

- **Run a specific test file**:
  ```bash
  npm test path/to/testFile.js
  ```

- **Generate coverage report**:
  ```bash
  npm run coverage
  ```
